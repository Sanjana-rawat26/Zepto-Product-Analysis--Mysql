# Zepto Products Analysis Using MySQL

## Project Overview

This project focuses on analyzing Zepto's product dataset using MySQL to extract meaningful business insights. The dataset contains 3,733+ product records across multiple categories. Through SQL-based analysis, pricing trends, discount patterns, inventory distribution, and category performance were explored to understand key business metrics and support data-driven decision-making.

##  Objectives

- Analyze product distribution across categories.
- Identify categories offering the highest discounts.
- Study pricing trends and selling prices.
- Evaluate inventory and stock availability.
- Generate business insights using SQL queries.
- Perform data exploration and aggregation analysis.
